# ui

npm 版本 >= 8.5.0

##  编译
首次编译,更换阿里的npm registry镜像
```
npm config set registry https://registry.npm.taobao.org
```
### 安装依赖
```
npm install
```

### 本地运行
```
npm run dev
```

### 静态文件
```
npm run build:prod
```
